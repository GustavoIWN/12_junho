# Gerador de Senhas

Projeto de criação de uma página web simples para gerar senhas com características personalizáveis.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Funcionalidade

O gerador permite escolher quais características a senha gerada deve ter:
- Letras maiúsculas
- Letras minúsculas
- Números
- Símbolos
